# LabTIC - Huffman File Compressor

## Overview

LabTIC is a C-based command-line application that implements the Huffman coding algorithm for lossless data compression. This project showcases Min-Heaps and binary trees to efficiently reduce file sizes without losing any underlying information.

## Key Features

- **Efficient Compression & Decompression:** Reliably reduces file sizes using the proven Huffman algorithm and restores them to their exact original state.
- **Interactive CLI:** An interface that makes it easy to compress, decompress, and analyze files directly from the terminal.
- **Data Visualization:** Inspect file contents in multiple formats, including Hexadecimal, Binary, and Raw Characters.
- **Size Comparison:** Built in tools to compare original and compressed file sizes.

### Usage

Run the compiled `LabTIC` executable. You will be greeted with an interactive menu offering the following options:

1. Load and display a file
2. Compress a file
3. Decompress a file
4. Compare sizes (Hexadecimal)
5. Compare sizes (Binary)
6. Compare sizes (Characters)
7. Exit

## About This Project

This project was made to deepen the understanding of fundamental computer science concepts, specifically data compression algorithms and low-level memory management in C.
